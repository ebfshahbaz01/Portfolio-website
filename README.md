# Portfolio Website

This repository contains a personal portfolio website built with React and Vite. It includes multiple sections such as Home, About, Skills, Portfolio, and Contact, with smooth navigation using React Router.

## Features

- Responsive single-page portfolio layout
- Section-based navigation using React Router
- Font Awesome icons powered by the React Font Awesome package
- Fast development and build workflow with Vite

## Tech Stack

- React
- Vite
- React Router DOM
- React Router
- Font Awesome React components

## Main Dependencies

This project uses the following packages:

- react-router-dom
- react-router
- @fortawesome/react-fontawesome
- @fortawesome/fontawesome-svg-core
- @fortawesome/free-brands-svg-icons
- @fortawesome/free-regular-svg-icons
- @fortawesome/free-solid-svg-icons

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ebfshahbaz01/Portfolio-website.git
   cd Portfolio-website
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and visit:
   ```text
   http://localhost:5173
   ```

## Available Scripts

- `npm run dev` - starts the development server
- `npm run build` - creates a production build
- `npm run preview` - previews the production build locally
- `npm run lint` - runs ESLint

## Project Structure

- src/Components - page and UI components
- src/Routes - route-related files
- public/ - static assets
- src/assets - images and other assets

## Notes

If you want to customize the portfolio, you can update the component files in the src/Components folder and adjust the routing in the route configuration.
